## ❓ 如何运行游戏

1. 确保已安装 `Python` 和 `Pygame` 库
2. 克隆或下载本仓库的代码文件
3. 在终端或命令行界面中，进入代码文件所在的目录
4. 运行以下命令启动游戏：

   ```shell
   python Game.py
   ```

## 🎮 游戏操作说明
- 按下空格键或向上键：跳跃
- 按住向下键：下蹲
- 松开向下键：恢复站立状态

 ## 👶 如何切换自己心仪的角色
- 将图片文件移动到Character-Rush-main\Character-Rush-main\resources\images中
- 将core.py文件中的ch项更改为自己的图片的路径即可使用自己心仪的角色
